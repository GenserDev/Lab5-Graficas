## Requisitos

## ¿Como ejecutar el programa? 
1. Clonar el repositorio.
```
git clone https://github.com/GenserDev/Lab5-Graficas.git
```
2. Entrar a la carpeta del repositorio.
```
cd Lab5-Graficas
```
3. Abrir su compilador de C y C++ y ejecutar los comandos

- Para compilar
```
cargo run --release
```
